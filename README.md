# 📊 Excel Visual Display Dashboard Project

This project showcases multiple business visualization exercises created in Microsoft Excel.

## 🔍 Project Highlights
- Sales trend analysis
- Customer preference insights
- Competitor benchmarking
- Budget vs actual variance
- Demographic segmentation
- Dashboard storytelling

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Radar Charts
- Line Charts
- Bar Charts
- Variance Analysis

## 📈 Key Business Use Cases
- Monthly sales trend reporting
- Gender-based car sales comparison
- Favourite film demographic analysis
- Competitor positioning
- Budget performance tracking
